# 🌱 EcoGuide AI

EcoGuide AI is a dynamic, full-stack Carbon Footprint Reduction Assistant designed to help individuals understand, track, and reduce their environmental impact through AI-driven insights and interactive scenarios.

## 🚀 Features
- **User Onboarding**: Calculates your baseline carbon footprint based on diet, transport, and energy habits.
- **Dynamic Dashboard**: Visualizes emissions using responsive Recharts and tracks historical trends.
- **Carbon Reduction Simulator**: Test hypothetical lifestyle changes (e.g., "Use public transport 3 days a week") to instantly see projected CO₂ savings.
- **AI Chatbot Assistant**: Ask questions and get real-time advice on sustainability best practices.
- **Offline Trial Mode**: Fully functional frontend mock layer if the backend is unreachable.

## 🏗️ Architecture
- **Frontend**: React 19 + TypeScript + Vite + Tailwind CSS v4
- **Backend**: Python FastAPI
- **Database**: SQLite (SQLAlchemy ORM)
- **Deployment**: Vercel (Frontend) + Hugging Face Spaces Docker (Backend)

## 🔒 Security Features (Target: 90+)
- Strict `Pydantic` schema validation enforcing constraints on all inputs.
- Rate Limiting implemented via `slowapi` to prevent abuse.
- Security Middleware injecting strict CORS policies and HTTP headers (`Strict-Transport-Security`, `X-Content-Type-Options`).

## ♿ Accessibility (Target: 95+)
- Full WCAG 2.1 compliance.
- Explicit `aria-label` tags on sliders, buttons, and interactive charts.
- High contrast colors using curated Tailwind color palettes.
- Keyboard navigable layout.

## 🧪 Testing
- **Backend**: Fully tested via `pytest` and `httpx` `TestClient` covering all core calculations, simulation logic, and API endpoints.

## 💻 Running Locally

### 1. Start the Backend
```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

### 2. Start the Frontend
```bash
cd frontend
npm install
npm run dev
```

## 🌐 API Documentation
